 # BiKE-SPHOTA with PubChem Compound Enhancement                                                                                                                                       
                                                                                                                                                                                        
  Extension of [BiKE-SPHOTA](https://github.com/bharathchand10/BiKE-SPHOTA) with chemical ontology enrichment using PubChem data.                                                       
                                                                                                                                                                                        
  ## What's New   

  This enhancement adds RDKit chemical property triples to the knowledge graph, improving compound embeddings in K-BERT training.                                                     
   
  ## Steps

  Replace BiKE-SPHOTA's step 4.16 (phrases_and_triples.ipynb) with the file in this repository with the same name - phrases_and_triples.ipynb
  Remaining steps are the same.
